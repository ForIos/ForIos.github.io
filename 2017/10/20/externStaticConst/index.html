static,const和extern

过程中学到的相关知识点:
@()[ iOS定义静态变量、静态常量、全局变量]

参考相关博客
变量

静态变量

作用域 不仅仅是作用域某个类的某个对象,而是作用域整个类的时候，这时候就可以使用静态变量。

static

在C和Java中用static关键之声明,可以用类名.变量使用
但是在OC中在程序启动时,只执行一次

静态常量const

几种常见的写法:
变量:
static const NSTimerInterval timeDuration = 0.5
对象:
static NSString const lmjName = @’’lmjName”;
static const NSString lmjName2 = @”lmjName2”;
这两个对象还是可以修改的,const 修饰的是LMJName,指的意思是内存地址不变的
正确写法
static NSString const LMJName = @”lmjName”;
不可以修改

全局变量extern

全局变量只能被初始化一次
